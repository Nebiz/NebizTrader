# NebizTrader
**POE 2 Simple Price Checker**

- Use **Ctrl+D** to open the item on the poe2 trade website.  
- This is a work in progress, functionning prototype.

### How to install
 - Install AHK v2.0 (not 1.1)- https://www.autohotkey.com/
 - Run script
 - If you don't want to install AHK, you can use the .exe

### Important Notes:

- If it's ever bugged, press **F12** to shutdown.
- This script automatically close **5 seconds** after **POE** is closed.
- **AHK** can be flagged by anti-cheats, be **careful** when using it **with other games**.
